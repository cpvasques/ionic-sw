NPM para instalar:
npm install cordova ionic
npm install ionic

1º - Para criar a aplicacão em ionic:
ionic start nomeProjeto blank --type ionic1

Blank é o template que será criado em branco.
Tipos de template padrão:
https://ionicframework.com/docs/v1/getting-started/

2º - Subir server WEB:
ionic serve

O server subirá na porta padrão: 8100

3º - Subir o laboratório:
ionic serve --lab
